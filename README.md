# Kuwo music provider for FeelUOwn player

## Installation

### Arch Linux
`pacman -S feeluown-kuwo`

### Install from pypi
`pip install --user feeluown-kuwo`

### Install with pip
`pip install --user .`

## Development with pipenv
`pipenv install` to create venv and start development.

## Run tests
`pytest -v`

test_kuwo_api.py -- Kuwo API tests
